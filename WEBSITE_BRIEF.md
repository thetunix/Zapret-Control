# Zapret Control Center — Описание программы для разработки сайта

## Общая информация

**Zapret Control Center** — это графический интерфейс (Electron-приложение) для управления инструментом **Zapret** на Windows. Zapret — это утилита для обхода блокировок интернета, работающая с Discord и YouTube.

Программа позволяет пользователю легко управлять различными конфигурациями Zapret без работы с командной строкой.

---

## Основной функционал

### 1. **Управление конфигурациями**
- Запуск, остановка, перезапуск процесса `winws.exe` (основная утилита Zapret)
- Выбор из 17 предустановленных конфигураций (general.bat, general (ALT).bat, general (SIMPLE FAKE).bat и т.д.)
- Каждая конфигурация использует разные методы обхода блокировок
- Текущая конфигурация отображается в интерфейсе

### 2. **Автоматическое тестирование конфигураций**
- Тестирует все конфигурации против целевых сайтов/сервисов
- Использует файл `targets.txt` со списком целей для проверки
- Находит и запоминает "лучшую конфигурацию" для пользователя
- Результаты тестов выводятся в реальном времени (название конфига, прогресс)
- Поддерживает разные режимы тестирования (fast, full и т.д.)

### 3. **Управление списками исключений**
- Пользователь может редактировать список `list-general-user.txt`
- Позволяет добавлять/удалять сайты из фильтрации
- Изменения применяются динамически

### 4. **Автоматические обновления**
- Проверяет наличие обновлений Zapret на GitHub (репозиторий `Flowseal/zapret-discord-youtube`)
- Скачивает и применяет обновления автоматически или по запросу
- Отслеживает версии

### 5. **Автозапуск с Windows**
- Может запускаться при старте ОС
- Автоматически включает лучший найденный конфиг
- Работает в фоновом режиме (режим трея)

### 6. **Режим трея**
- При сворачивании программа переходит в системный трей
- Быстрый доступ к управлению из трея
- Может свернуться в фон при старте (опция `startMinimized`)

### 7. **Логирование**
- Ведёт логи всех действий программы
- Хранит до 500 последних записей в памяти
- Логи отображаются в интерфейсе
- Может очищаться пользователем

### 8. **Фильтр игр**
- Опция `gameFilterMode` (disabled, enabled или другие режимы)
- Может исключить трафик игр из фильтрации

### 9. **Визуальное оформление**
- 9 тем (Figma Dark, Aurora, Graphite, Ember, Glacier, Orchid, Forest, Ruby, Noir)
- Параметры: `glassBlur` (размытие фона), `ambientIntensity` (интенсивность), `motionSpeed` (скорость анимаций)
- Glass-морфизм стиль с живыми фонами

### 10. **Нужны права администратора**
- Для запуска `winws.exe`, тестирования и управления сервисом Windows требуются права администратора

## Структура файлов программы

```
zapret/
├── bin/
│   └── winws.exe              # Основная утилита (процесс, который запускает программа)
├── general.bat                # Конфигурация по умолчанию
├── general (ALT).bat          # Альтернативные конфигурации (17 вариантов)
├── general (FAKE TLS AUTO).bat
├── ...
├── service.bat                # Управление сервисом Windows
├── lists/                      # Списки для фильтрации
│   ├── list-general.txt       # Базовый список целей
│   ├── list-general-user.txt  # Пользовательский список (редактируется в UI)
│   ├── list-google.txt
│   ├── list-exclude.txt       # Сайты для исключения из фильтрации
│   └── ipset-all.txt          # IP-адреса для фильтрации
└── utils/
    └── targets.txt            # Целевые сайты для тестирования конфигов
```

## Параметры и настройки (хранятся в памяти программы)

```javascript
{
  selectedConfig: 'general.bat',     // Текущий выбранный конфиг
  bestConfig: 'general.bat',         // Лучший конфиг (найденный при тестировании)
  theme: 'figma',                    // Текущая тема оформления
  launchAtLogin: true,               // Запускаться при старте Windows
  autoStartZapret: true,             // Автоматически запускать Zapret
  autoUpdateZapret: true,            // Автоматически обновляться
  startMinimized: false,             // Запускаться свёрнутой
  gameFilterMode: 'disabled',        // Режим фильтра для игр
  testMode: 'fast',                  // Режим тестирования
  ambientIntensity: 72,              // Интенсивность фона (0-100)
  glassBlur: 18,                     // Размытие фона (пиксели)
  motionSpeed: 55                    // Скорость анимаций (%)
}
```

## События и состояния

Программа обновляет UI в реальном времени через события:

- **State Update** (каждые 3 сек) — статус winws.exe, текущий конфиг, версия
- **Logs** — новые логи записываются и отображаются
- **Update Progress** — статус проверки и установки обновлений
- **Tests** — начало, прогресс, результаты тестирования
- **Status Changes** — конфиг включен/выключен, сервис запущен/остановлен
