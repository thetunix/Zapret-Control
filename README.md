# Zapret Control Center

Electron-панель управления для локального набора
[`Flowseal/zapret-discord-youtube`](https://github.com/Flowseal/zapret-discord-youtube)
на Windows. Приложение помогает запускать, останавливать, тестировать и обновлять
`zapret` без ручной работы с `.bat`-файлами и командной строкой.

## Возможности

- запуск, остановка и перезапуск `zapret/bin/winws.exe` из графического интерфейса;
- выбор конфигураций `zapret/general*.bat`;
- установка Windows-службы `zapret` для запуска без UAC при каждом старте;
- автозапуск приложения вместе с Windows;
- автоматическое включение лучшего найденного конфига;
- проверка и применение обновлений `zapret`;
- тестирование конфигов по целям из `zapret/utils/targets.txt`;
- редактирование пользовательского списка доменов `zapret/lists/list-general-user.txt`;
- логи, трей-режим, темы интерфейса и настройка игрового фильтра.

## Как работает режим без постоянного UAC

Windows не позволяет приложению тихо получить права администратора при каждом
запуске. Поэтому Control Center использует нормальную схему через службу:

1. Откройте приложение обычным способом.
2. Перейдите в `Настройки` -> `Служба Windows`.
3. Нажмите `Установить`.
4. Подтвердите UAC один раз.

После этого создается служба `zapret`, которая запускает `winws.exe` с нужными
правами. Обычные кнопки включения и выключения используют уже установленную
службу, поэтому UAC больше не появляется при каждом запуске.

Если выбран другой конфиг, приложение переустановит службу с новыми аргументами.
Для этого снова потребуется подтверждение UAC, потому что меняется системная
служба Windows.

## Требования

- Windows 10 или Windows 11;
- Node.js 18+ для разработки;
- установленный `npm`;
- папка `zapret/` рядом с приложением, включая `bin/winws.exe` и `service.bat`.

## Запуск для разработки

```powershell
npm install
npm start
```

Явный запуск с запросом прав администратора:

```powershell
npm run start:admin
```

Обычный `npm start` не вызывает UAC автоматически. Это удобно для разработки
интерфейса и не закрывает dev-сессию при отказе от повышения прав.

## Сборка

```powershell
npm run build
```

Готовый NSIS-инсталлятор появляется в папке `dist/`.

## Структура

```text
.
├── src/
│   ├── main.js       # Electron main process и управление zapret
│   ├── preload.js    # безопасный IPC bridge
│   ├── renderer.js   # логика интерфейса
│   ├── index.html    # окно приложения
│   └── styles.css    # стили интерфейса
├── zapret/           # локальные файлы Flowseal zapret-discord-youtube
├── package.json
└── README.md
```

## Релиз

Версия задается в `package.json`. Перед релизом:

```powershell
npm run build
```

Основной артефакт для публикации:

```text
dist/Zapret Control Center Setup <version>.exe
```

## Важно

Проект является графической оболочкой для локального `zapret` и не является
официальным релизом `Flowseal/zapret-discord-youtube`.
